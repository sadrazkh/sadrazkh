# 🧪 Sadra's Product Lab

I build software that solves real problems.

Most of my projects start with a simple question:

> **"Why am I still doing this manually?"**

If something wastes time, creates friction, or lacks the features I need, I try to build a tool for it.

---

## ⚡ What I Build

```txt
Real Problem
    ↓
Practical Tool
    ↓
Daily Usage
    ↓
Better Workflow
```

- 🧰 Internal tools
- 🖥 Desktop applications
- ⚙️ Automation systems
- ☁️ Self-hosted platforms
- 🛰 VPN / Xray tooling
- 📚 AI-assisted learning tools
- 🏢 Business management software

## 🧠 Engineering Philosophy

```txt
I don't build projects just because they are cool.

I build them when they remove friction.
```

### I care about:

- ✅ Real usage over fake portfolio projects
- ✅ Practical UX over unnecessary complexity
- ✅ Automation over repetitive work
- ✅ Shipping usable tools
- ✅ Understanding the problem before choosing the stack
